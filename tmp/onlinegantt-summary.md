## Overview
OnlineGantt is a browser-based Gantt chart editor. The page emphasizes:
- no signup
- no download required
- free use
- local `.gantt` file support
- a cloud option for saving and sharing with a team

## What the interface offers
### Project/file actions
- New
- Open `.gantt` file
- Save As `.gantt` file
- Import / Export
- Export to Excel, PDF, and image files

### Planning controls
- Today
- All Tasks
- Remaining Tasks
- Overdue Tasks
- Zoom in
- Zoom out
- Zoom to fit
- Expand all / Collapse all

### Views and settings
- Project View
- Resource View
- Edit Resources
- Edit Settings
- Share
- Revisions
- Autosave toggle

### Task operations
- Add
- Edit
- Delete
- Indent / Outdent
- Dependency handling
- Validation of editing
